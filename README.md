# Task 4: SQL for Data Analysis

## Objective
To extract, manipulate, and analyze structured e-commerce data using SQLite to demonstrate foundational SQL analytical capabilities.

## Deliverables Included
* **Task4_Queries.sql**: The complete SQL script containing data generation schemas and analytical queries.
* **Screenshots**: Documentation of execution results across all targeted requirements (Filtering, Sorting, Joins, Subqueries, Views, and Indexes).

## Core Concepts Applied
1. **Filtering & Sorting**: Used `WHERE`, `GROUP BY`, and `ORDER BY` to segment high-value customer order behaviors.
2. **Table Joining**: Connected relational data via `INNER JOIN` structures between user and order tables.
3. **Subqueries**: Leveraged nested selection logic to dynamically isolate product entries priced above the inventory average.
4. **Aggregations**: Summarized metrics via `COUNT()`, `SUM()`, and `AVG()`.
5. **Optimization**: Constructed specialized `VIEW` queries for saved summaries and generated query tables using target `INDEX` fields to maximize lookup speeds.
